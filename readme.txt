1. Please unzip the file by the given password

2. To build the program, after unzip the folder, open terminal and direct to the following path
> cd ${unzip directory}/MorganStanleySpring-main
> ./gradlew bootrun

Remark: please note that the application is using localhost on port 8080

3. Open the console from the following link after the application start successfully
- http://localhost:8080/

Remark: the Mock trade file is located under ${unzip directory}/MorganStanleySpring-main/src/main/resources/sample

4. Please feel free to ask me if there is any problems
